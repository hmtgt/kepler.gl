# Demo App

This is the src code of kepler.gl demo app.


### Local dev
```
npm install
```
or
```
yarn --ignore-engines
```

add mapbox access token to node env
```
export MapboxAccessToken=<your_mapbox_token>
```

add google api key to node env
```
export GoogleApiKey=<insert_your_key>
```

then
```
npm start
```
